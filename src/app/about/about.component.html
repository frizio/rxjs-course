Stream (of values)

Everything almost is asynchronous. We have requests coming from the network bringing new values from the backend,we have timeouts occurring in the frontend, we have user interaction with click mouse over events. All of those are asynchronous events that we need to combine in order to produce the final result of our program.

  • User interactions
  • Backend requests
  • Timeout intervals

RxJS Core use case: combine multiple streams

RxJS just stands for reative extensions for javascript. It's a library that makes it very simple to combine streams of values together in a maintainable way.
